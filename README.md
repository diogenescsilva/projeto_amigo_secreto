# 🎲  Challenge Amigo secreto 

Este é um projeto simples em HTML, CSS e JavaScript, desenvolvido do curso da Oracle One, que permite adicionar nomes a uma lista, sortear aleatoriamente um deles e removê-lo da lista após o sorteio.  

## 🚀 Funcionalidades  
- Adicionar nomes à lista  
- Exibir os nomes adicionados  
- Sortear aleatoriamente um nome  
- Remover o nome sorteado da lista  
- Exibir um alerta caso a lista esteja vazia  

## 📌 Como Usar  
1. Digite um nome no campo de entrada.  
2. Clique no botão **"Adicionar"** para inserir o nome na lista.  
3. Clique no botão **"Sortear Amigo"** para sortear um nome aleatório.  
4. O nome sorteado será exibido e removido da lista automaticamente.  
5. Repita o processo até que todos os nomes tenham sido sorteados.  

## 🛠️ Tecnologias Utilizadas  
- HTML  
- CSS 
- JavaScript
